# Installation
Require package:
```bash
composer require stephenlake/nova-fixed-bars
```

Publish and edit configuration file (**Optional**)
```bash
php artisan vendor:publish --provider="NovaFixedBars\NovaFixedBarsServiceProvider" --tag="config"
```
